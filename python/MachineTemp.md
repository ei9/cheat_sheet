# 熱當機

## Question

T(i) = T(i - 1) + i * 2.71828

## Input
**n** : 有多少筆資料 (1 <= n <= 10)

**T(0)** : 初始溫度，為正實數 (1 <= T(0) <= 100)

**i** : 加熱時間，為正整數 (1 <= i <= 100)

## Tips:
觀察一下：

**T(3)** = **T(2)** + i * 2.71828

**T(2)** = **T(1)** + i * 2.71828

**T(1)** = **T(0)** + i * 2.71828

> 我們宣告一個函式T，將初始值***T0***以及目前時間***i***作為參數

> ***T(t0, i)***

這個函式將會呼叫**自己**，取得前一秒的溫度T(i-1)。

這樣子i要減1。

i一直減1最後變成**零**，就回傳**T(0)—初始值**，不然就回傳**T(i - 1)**，這就是**遞迴**。
