# 熱當機

## Question

T(i) = T(i - 1) + i * 2.71828

## Input
**n** : 有多少筆資料 (1 <= n <= 10)

**T(0)** : 初始溫度，為正實數 (1 <= T(0) <= 100)

**i** : 加熱時間，為正整數 (1 <= i <= 100)

## Tips:
**T(3)** = **T(2)** + i * 2.71828

**T(2)** = **T(1)** + i * 2.71828

**T(1)** = **T(0)** + i * 2.71828

這題用遞迴，如果用動態規劃法會更快。
